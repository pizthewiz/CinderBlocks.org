A CinderBlock is a containerized collection of code and/or libraries that extend the C++ library Cinder. The official CinderBlock docs and the forum thread RFC: CinderBlocks offer more detail on the initial motivitations, capabilities and the limitations of CinderBlocks.

The source for the CinderBlocks.org project is available on GitHub.
